```go
package main
 
import (
    "fmt"
)
 
func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}
 
 
func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}
 
 
func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}

Программа выведет :
2
1

Сначала выводится рез-т функции test, в кот-ой объявляется перем-я x со значение 1, затем вызывается отложенная анон-я функции,
внутри кот-й инткрементируется значение x, и после этого возвращается рез-т

В случае с функцией anotherTest, объявляется целочисленная перем-я, затем ей присваивается значение 1, после этого возвращается как значение
и отложенная анонимная функция отрабатывает только после вовзврата x
