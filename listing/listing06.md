```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s) 
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}

Программа выведет [3 2 3]
В main создаем слайс s. Передаем его как значение в функцию modifySlice().
Срезы в Go имеют базовый массив и передаются по ссылке, а не по значению. Мы просто передаем ссылку на срез в функцию, затем меняем ее, и она меняет внешний срез. Поэтому i[0] = "3" изменит срез s в main().
Далее добавляем один элемент в срез внутри функции, но это поменяет срез. Создается новый срез, ccылающийся на новый базовый массив, И обычно удваивает объем базового массива вдвое, поэтому дальнейшие изменения в срезе будут видны только внутри функции modyfySlice.

Когда мы хотим добавить еще один элемент, у нас нет места для него в базовом массиве. Затем встроенная функция добавления создает новый массив и новый срез. А тогда новый срез будет ссылаться на др область памяти
